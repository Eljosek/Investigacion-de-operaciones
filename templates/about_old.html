¿Qué es la Programación Lineal?

La programación lineal es una técnica de optimización matemática que busca encontrar el mejor resultado (máximo o mínimo) de una función objetivo lineal, sujeta a un conjunto de restricciones lineales.

Función Objetivo

Expresión matemática que queremos optimizar:

Maximizar: Z = c₁x₁ + c₂x₂ + ... + cₙxₙ
Minimizar: Z = c₁x₁ + c₂x₂ + ... + cₙxₙ
Restricciones

Limitaciones que debe satisfacer la solución:

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≥ b₂
x₁, x₂, ..., xₙ ≥ 0 (no negatividad)

Formato de Entrada

Función Objetivo
Ejemplos válidos:
maximizar z = 3x1 + 2x2
minimizar z = 5x1 + 4x2 + 3x3
max z = x1 + 2x2 - x3
Restricciones
Ejemplos válidos:
x1 + x2 <= 4
2x1 + 3x2 >= 6
x1 + 2x2 + x3 = 10
Nota: Las restricciones de no negatividad (xi ≥ 0) se agregan automáticamente y no necesitan especificarse.

    • El Método Gráfico

    El método gráfico es una técnica visual para resolver problemas de programación lineal con dos variables. Los pasos son:

    Graficar las restricciones

    Cada restricción se representa como una línea recta en el plano cartesiano.

    Identificar la región factible

    Es el área que satisface todas las restricciones simultáneamente.

    Encontrar los vértices

    Los puntos de intersección de las líneas que forman las esquinas de la región.

    Evaluar la función objetivo

    El óptimo siempre se encuentra en un vértice de la región factible.

    Ventajas

    • Fácil de entender visualmente
    • Ideal para problemas educativos
    • Muestra la región factible claramente
    • Permite análisis de sensibilidad

    Limitaciones

    • Solo funciona con 2 variables
    • Impractical para problemas grandes
    • Requiere que todas las restricciones sean lineales
    • La precisión depende de la escala

    Formato de Entrada

    Función Objetivo:
    • maximizar z = 3x + 2y
    • min z = x - 2y
    • max z = 5x + 4y
    Restricciones:
    • x + 2y <= 8 (menor o igual)
    • 2x + y >= 6 (mayor o igual)
    • x + y = 5 (igualdad)
    • x >= 0 (no-negatividad)
    Consejo: Las restricciones de no-negatividad (x ≥ 0, y ≥ 0) se incluyen automáticamente.
