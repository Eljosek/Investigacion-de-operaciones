Método Simplex Dos Fases

Resuelve problemas con restricciones mixtas (<=, >=, =) usando variables artificiales

Ingresa tu problema

Formatos: "max z = 3x1 + 2x2" o "min z = 5x1 + 4x2 + 3x3"
Ideal para: Restricciones >=, = o mixtas (requiere variables artificiales)
¿Cuándo usar Dos Fases?
  • Cuando tienes restricciones >= o =
  • Cuando el Simplex estándar no puede iniciar (no hay solución básica obvia)
  • Para problemas con variables artificiales
¿Cómo funciona el Método Dos Fases?

Fase I: Minimiza la suma de variables artificiales para encontrar una solución básica factible.

Fase II: Optimiza la función objetivo original usando la solución encontrada en Fase I.

Si Fase I termina con W=0, el problema es factible y continúa a Fase II. Si W>0, el problema es infactible.
